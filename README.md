# SkyCast Pro 🌤️

A modern, responsive **Weather Dashboard** developed using **HTML5, CSS3, and Vanilla JavaScript** for **Prodesk IT Sprint 03**.

SkyCast Pro provides real-time weather information using the **OpenWeatherMap API** and demonstrates modern frontend development concepts including asynchronous JavaScript, API integration, geolocation, caching, and responsive UI design.

## 📌 Project Objectives

* Integrate a live weather API.
* Learn asynchronous JavaScript using `fetch()` and `async/await`.
* Display live weather information.
* Handle API errors gracefully.
* Implement browser geolocation.
* Improve user experience using animations and responsive design.
